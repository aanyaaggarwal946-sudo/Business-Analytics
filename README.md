# Business-Analytics
 SBI Business Analytics Project

## Project Overview
This project focuses on analyzing State Bank of India (SBI) financial data using Python for Business Analytics. Various data visualization and statistical techniques were used to identify trends, relationships, correlations, and financial insights from SBI datasets.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Project Modules

| Analysis | Description |
|---|---|
| Total Assets Trend Analysis | Visualized SBI’s total assets growth over the years using line charts. |
| Borrowings vs Total Assets | Analyzed the relationship between borrowings and total assets using regression analysis. |
| Distribution of Reserves | Studied reserve distribution patterns using histogram and KDE plots. |
| Fixed Assets vs Investments | Compared fixed assets and investments using box plot visualization. |
| Asset Deployment Category Analysis | Classified asset deployment into Low Deploy and High Deploy categories. |
| Equity Capital vs Reserves | Visualized reserve growth trends using bar charts. |
| Correlation Heatmap | Identified correlations among financial variables through heatmaps. |
| Asset Mix Analysis (2025) | Analyzed SBI’s asset composition for the year 2025 using pie charts. |
| Borrowings vs Liabilities by TAD | Compared liabilities and borrowings based on TAD categories. |
| Correlation-Based Financial Insight | Evaluated leverage and debt dependency using correlation analysis. |

## Key Insights
- SBI’s total assets showed consistent growth over the years.
- Strong positive correlation observed among investments, reserves, and total assets.
- Borrowings significantly influenced liabilities and overall asset growth.
- Asset deployment patterns highlighted different financial deployment categories.
